## Arquitetura

O módulo Java é responsável pela lógica principal do ROTTA, realizando o gerenciamento de usuários, validação das regras de negócio, controle de postagens e gerenciamento da carteira digital.

<br>

| Camada | Responsabilidade |
|---|---|
| Model | Estrutura das entidades |
| Service | Regras de negócio |
| Main | Execução do sistema |
